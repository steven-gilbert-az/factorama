Detailed Description

Generic linear prior constraint for linear variable types.

Applies linear prior constraints by penalizing deviations from expected values. Use only with linear variables (LandmarkVariable, GenericVariable) - not for rotations or poses which require manifold-aware constraints.

auto landmark_prior = std::make_shared<GenericPriorFactor>(
factor_id++, landmark, Eigen::Vector3d(0.0, 0.0, 5.0), 1.0);
