Detailed Description

Sparse non-linear least squares optimizer.

Implements Gauss-Newton and Levenberg-Marquardt algorithms using Eigen's sparse linear algebra. Designed to work with FactorGraph for bundle adjustment and SLAM problems.

SparseOptimizer optimizer;
OptimizerSettings settings;
settings.method = OptimizerMethod::LevenbergMarquardt;
optimizer.setup(graph_ptr, settings);
optimizer.optimize();
