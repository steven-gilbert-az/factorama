Detailed Description

3D landmark variable in Euclidean space

Represents 3D world landmarks as simple position vectors. Commonly used for bundle adjustment problems with camera observations.

Eigen::Vector3d landmark_pos(0.0, 1.0, 6.0);
auto landmark = std::make_shared<LandmarkVariable>(2, landmark_pos);
