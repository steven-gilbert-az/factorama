Detailed Description

Prior constraint for rotation variables using SO(3) manifold.

Applies manifold-aware prior constraints to RotationVariable instances. Uses proper SO(3) manifold operations.

Eigen::Matrix3d prior_rotation = Eigen::Matrix3d::Identity();
auto rotation_prior = std::make_shared<RotationPriorFactor>(
factor_id++, rotation_var, prior_rotation, sigma);
