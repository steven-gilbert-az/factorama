Detailed Description

Prior constraint on camera position.

Applies a prior constraint to the position component of a PoseVariable. Useful for anchoring poses or incorporating GPS measurements.

double position_sigma = 0.5;
auto pose_position_prior = std::make_shared<PosePositionPriorFactor>(
factor_id++, camera_pose, Eigen::Vector3d::Zero(), position_sigma);
