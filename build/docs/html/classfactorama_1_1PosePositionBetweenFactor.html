Detailed Description

Position-only relative constraint between pose variables.

Constrains the position difference between two poses to match a measured difference variable. Only affects the translation components of poses, ignoring rotation.

auto position_constraint = std::make_shared<PosePositionBetweenFactor>(
factor_id++, pose_a, pose_b, measured_delta_variable, sigma);
