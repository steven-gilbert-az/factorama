Detailed Description

Rotation-only variable for extrinsic calibration.

Represents rotations between coordinate frames using SO(3) manifold operations. Commonly used for camera-IMU calibration where only relative orientation matters.

Eigen::Matrix3d initial_rotation = Eigen::Matrix3d::Identity();
auto rotation_var = std::make_shared<RotationVariable>(1, initial_rotation);
